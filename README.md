Teresia — Веб-сайт цветочного магазина

Описание проекта

Веб-сайт цветочного магазина, созданный в учебных целях. 
Позволяет пользователям просматривать каталог товаров, добавлять их в корзину и переходить к оформлению заказа.
Дизайн вдохновлён мотивами Нью‑Йорка, фокус на визуальную подачу и удобную навигацию. 
Адаптивная верстка не реализована.

Система предоставляет возможность регистрации и авторизации пользователей. 
Каждый пользователь получает личный кабинет, где может редактировать свои личные данные и просматривать историю покупок.
Реализован полный цикл управления товарами: добавление, просмотр списка, редактирование и удаление.
Каждый товар имеет свои категорию, цену, описание, фотографии. Управление происходит с помощью панели администратора на которой отображается статистика по заказам
с возможностью выгрузки информации за определенный период в документ Excel,
меню для управления товарами, представленными на сайте.
Распределение товаров по категориям обеспечивает удобную организацию и быстрый поиск.

Технологии

Backend: PHP, MySQL (через phpMyAdmin)
Frontend: HTML, CSS, JavaScript
Tools: VS Code, Git, локальный сервер AMPPS
